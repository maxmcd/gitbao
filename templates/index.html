gitbao

Welcome to gitbao. Gitbao allows you to easily deploy an application from a github gist.

Take some code: 
package main

import "net/http"

func main() {
    http.HandleFunc("/",
        func(w http.ResponseWriter, req *http.Request) {
            w.Write([]byte("Gitbao is awesome"))
        })
    http.ListenAndServe(":8080", nil)
}
Create a github gist for it:

https://gist.github.com/maxmcd/f430e80ab3246b2ae6d9

Change the url:

https://gist.gitbao.com/maxmcd/f430e80ab3246b2ae6d9

That's it.


Gitbao is currently very limited. It only works with Go applications, and running applications can be killed at any time.
Other language support is on the way, along with better application control, auto-updates, logins, etc...
